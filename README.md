# user_api_spring_rest_postgres_docker
This project contains REST API and CLIENT.<br/>

<b>Required programs:</b>
- Maven;
- Docker;
- Docker Compose;
- Java 8+;

<b>How to run?</b><br/>
./start.sh

./stop.sh

<b>Hosts:</b>
- http://localhost:8083/client (Client)
- http://localhost:8080/user_api (API)
- http://localhost:8081/adminer (Adminer)
- http://localhost:8082 (Postgres)
